# Calculadora de Precios — Orchid

Calculadora de precios simple para pedidos — interfaz estática con generación de factura en PDF.

Contenido del repositorio:
- `index.html` — interfaz principal
- `css/styles.css` — estilos
- `js/script.js` — lógica de la calculadora, generación de tarjetas y PDF
- `img/logo.png` — logo usado en el PDF

Uso rápido (local):
1. Abrir `index.html` en el navegador.
2. Ingresar precio en USD y pulsar "Agregar Producto" para crear tarjetas.
3. Pulsar "Generar Factura" para descargar el PDF.

Puedes subir este proyecto a GitHub siguiendo las instrucciones en la raíz del proyecto.
